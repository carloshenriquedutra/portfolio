Sobre este portfólio

Este site não é uma vitrine de ferramentas, mas um índice curado de projetos que exploram decisões arquiteturais reais em engenharia de dados: ingestão, processamento, armazenamento, custo, confiabilidade e evolução.

Cada projeto assume limitações explícitas e discute onde o sistema quebra.
